# JBC Homepage Redesign

A single-page prototype redesign of the [Jennifer Bett Communications](https://www.jenniferbett.com) homepage — concepted as a design exploration, not an official JBC deliverable.

## View the prototype

Live: **https://bczarny.github.io/jbc-homepage-redesign/**

*(Link goes live once GitHub Pages is enabled: Settings → Pages → Source: `main` / `/ (root)`.)*

## What's in here

- `index.html` — the full single-file prototype (HTML, CSS, and JS all inline)
- `jbc-logo-trimmed.png` — the JBC wordmark used in the header and footer

## Structure

The page covers the full homepage narrative:

1. **Hero** — positioning, founders, tagline
2. **Stats strip** — headline numbers
3. **Our Work** — six case study tiles with real brand imagery (OLIPOP, Parachute, theSkimm, Sloomoo Institute, Grove Collaborative, FASHIONPHILE)
4. **Why JBC** — senior-led, independent, accountable
5. **Testimonials** — client and editor quotes
6. **What We Do** — twelve services, sectors strip, and an Executive Speaking & Awards feature
7. **CTA + Footer**

All imagery is pulled directly from JBC's Squarespace CDN so the prototype renders with real brand visuals.

## Editing

It's a single HTML file — open it in any editor, save, refresh. No build step, no dependencies.
